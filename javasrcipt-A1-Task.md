Just a simple video explaining that to watch them, we need to know a little bit about programming in general. Also, it is an introduction to all the persons that have participated in this serie of video.
- Video 2:
JavaScript is a programming language designed to interact with elements of a web page. We need to know JavaScript because it is everywhere (number 1 most used programming language in the world)
- Video 3:
This video shows the difference between the client JavaScript and the Server JavaScript (NodeJS). JavaScript on a client is to interact with the user interface on a website (with the DOM) and on a server, it is used to build web services.
- Videos 4 and 5:
This videos gives us the tools we need to code with JavaScript (VSS, extensions, NodeJS, WSL) and shows us how to install everything.
- Video 6:
Shows the very basics of JS/NodeJS. Shows the console.log(), the constants and how to print what we want.
- Videos 7 and 8:
Shows how to comment in a JS file. We can use single line(//) or multilines(/* */) comments. We should use fonctions to comment a function instead of every lines.
- Videos 9 and 10:
Show the 3 ways of declaring a variable : var (non used anymore), let (a variable) and const (a constant). The main difference between var and let is that var is function scoped and let is block scoped.
- Videos 11 and 12:
Shows how to concatenate strings. We can use "+" between two strings to concatenate them. 
- Videos 13 and 14:
Template literals is another way to formate strings. It works by inserting ${string} in a sentence we want to print.
